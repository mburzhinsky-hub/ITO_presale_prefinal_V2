Паспорт проекта

v10
Ручной режим помогает инженеру собирать состав вручную, а система подсказывает сопутствующие позиции и считает монтаж/ПНР по отмеченным строкам.
Тип проекта влияет на набор рекомендуемых зон и на библиотеку оборудования.

Коэффициенты

1.00 — стандартный объект
1.00 — Москва/МО и простая доставка
Сложность
Влияет на монтаж, ПНР и часть сервисов. Растёт из-за высоты, количества зон, интерактива, VR, вау-сценариев, сжатых сроков и нестандарта.
Логистика
Влияет на поставку, кабельные/расходные и часть закупки. Растёт из-за удалённости региона, этажности, уличных работ и включённой доставки.

Сервисы по умолчанию

Левая колонка больше не должна наезжать на контент: она ограничена по высоте и нормально скроллится. Для GitHub Pages всё работает как один файл.

Механика просчёта

Сначала заносится паспорт проекта и зоны. Для каждой зоны указываются назначение, площадь, основная задача и нужные сервисы. Генератор сам подбирает стартовый состав решений по бюджету, а затем смету можно вручную править по брендам, моделям и ценам.
